## 5. Paint Girl
닌텐도 스플래툰과 모방하여 공부목적으로 만든 어플리케이션
페인트를 이용해 땅을 더 많이 칠한 사람이 승리하는 게임

- 중요 구현 기술
  - Singleton 패턴을 이용
  - Json파싱을 이용하여 각 몬스터와 HP, 공격력등 적용
  - 파티클을 이용하여 바닥이 칠해지는 파티클 생성
  - Git을 이용한 협업
- 사용 기술 및 툴
  - VS 2015, Unity3D, Json
